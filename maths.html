Trigonometry

If $\sin \theta = \frac{3}{5}$, find $\tan \theta$.

My Mistake: Messed up the base calculation.

Smart Solution: Use triplet 3, 4, 5. Perpendicular = 3, Hypotenuse = 5, so Base = 4. $\tan \theta = \frac{P}{B} = \frac{3}{4}$.

Percentage

If A's income is 25% more than B, how much percent is B's income less than A?

My Mistake: Guessed 25% less.

Smart Solution: Formula: $\frac{R}{100+R} \times 100$. $\frac{25}{125} \times 100 = 20\%$.
